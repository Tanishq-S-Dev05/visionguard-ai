# VisionGuard AI 🛡️

Real-time workplace safety monitoring system using computer vision. Detects PPE violations, fall incidents, and restricted zone intrusions with instant SMS/email alerts.

## 🎯 Features
- **PPE Detection:** Hard hat, safety vest, gloves detection using fine-tuned YOLOv8
- **Fall Detection:** Real-time worker fall identification
- **Zone Intrusion:** Restricted area monitoring
- **Instant Alerts:** Twilio/SendGrid SMS & email notifications
- **Live Dashboard:** React-based monitoring dashboard
